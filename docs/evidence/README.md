# Evidence Index

This directory groups project evidence by delivery phase so screenshots, reports, and verification artefacts stay organised as the testing scope grows.

## Phases

- [phase-1/README.md](phase-1/README.md) — clean start and baseline setup evidence

Add each later phase here using the same structure:

- `phase-N/README.md` — summary of what the phase proves
- screenshots, reports, or exported evidence files stored alongside that README
